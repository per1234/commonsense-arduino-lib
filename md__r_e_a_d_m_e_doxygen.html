CommonSense Arduino Library

Reference Documentation

Have a look at the library documentation for more information. The API methods available to developers can be found on the CommonSense Developers Pages.

Usage

Let's get started by including the necessary libraries:

#include <Time.h>
#include <CommonSense.h>

We will need some information to connect and logon to CommonSense:

const char *username = "test";
const char *password = "098f6bcd4621d373cade4e832627b4f6";
const char *deviceName = "Arduino Uno";
const byte MAC[] = { 0xAA, 0xBB, 0xCC, 0xDD, 0xEE, 0xFF };

We are also going to store the sensor identifiers:

long temperature;
long light;
long pressure;

Now it's time to create an object so we can talk to CommonSense:

CommonSense sense = CommonSense(username,password,deviceName,MAC);

Now that's done, we should use CommonSense::begin() inside the setup() of the sketch. This function sets up ethernet and time to call CommonSense::login().

sense.begin();

With CommonSense::login() having saved the session id within the object, we can create the sensors in our CommonSense account.

temperature = sense.createSensor("Temperature", "T1 Temperature Sensor", "T1SEN");
humidity = sense.createSensor("Light", "L1 Light Sensor", "L1SEN");
pressure = sense.createSensor("Pressure", "P1 Pressure Sensor", "P1SEN");

We got our sensors up, now it's time to acquire data and upload it.

sense.uploadData(temperature, analogRead(A0));
sense.uploadData(light, analogRead(A1));
sense.uploadData(pressure, analogRead(A2));

Your data should now be visible in the sensor library.
