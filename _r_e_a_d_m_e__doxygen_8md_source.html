1 ##Installation##
2 The Arduino IDE has a built-in function for installing libraries. Go to **Sketch > Import Library... > Add Library...** and select the zip file or folder containing the library.
3 
4 If you're not using the Arduino Development Environment make sure the files are in a folder named 'CommonSense' and place the folder in your libraries folder.
5 
6 ##Reference Documentation##
7 Have a look at the [library documentation](http://senseobservationsystems.github.com/commonsense-arduino-lib/) for more information.
8 The API methods available to developers can be found on the [CommonSense Developers Pages](http://www.sense-os.nl/developers).
9 
10 ##Examples##
11 Sketch examples are included with this library. You can select one from **File > Examples > CommonSense**.

13 ##Usage##
14 Let's get started by including the CommonSense library. This will also include arduino core dependencies:
15 
16 ~~~{.c}
17 #include <CommonSense.h>
18 ~~~
19 
20 We will need some information to connect and logon to CommonSense:
21 
22 ~~~{.c}
23 const char *username = "test";
24 const char *password = "098f6bcd4621d373cade4e832627b4f6";
25 const char *deviceName = "Arduino Uno";
26 const byte MAC[] = { 0xAA, 0xBB, 0xCC, 0xDD, 0xEE, 0xFF };
27 ~~~
28 
29 We will also need to store the sensor identifier:
30 
31 ~~~{.c}
32 long pressure;
33 ~~~
34 
35 Now it's time to create an object so we can talk to CommonSense:
36 
37 ~~~{.c}
38 CommonSense sense = CommonSense(username,password,deviceName,MAC);
39 ~~~
40 
41 Now that's done, we should use CommonSense::begin() inside the setup() of the sketch.
42 This function sets up ethernet and time to call CommonSense::login().
43 
44 ~~~{.c}
45 sense.begin();
46 ~~~
47 
48 With CommonSense::login() having saved the session id within the object,
49 we can create the sensors in our CommonSense account.
50 
51 ~~~{.c}
52 pressure = sense.createSensor("Pressure", "P1 Pressure Sensor", "P1SEN");
53 ~~~
54 
55 We got our sensors up, now it's time to acquire data and upload it.
56 
57 ~~~{.c}
58 sense.uploadData(pressure, analogRead(A0));
59 ~~~
60 
61 Your data should now be visible in the sensor library.
